v2.0.13 — Cross-part Takeout enrichment, photo captions, and a smarter By-Date timeline.

This release adds three substantial features plus several robustness fixes from real-library testing.

Cross-part Takeout enrichment:

Google's multi-part Takeouts split photos and their JSON sidecars across different ZIP parts. If photo IMG_1234.jpg is in part 7 and its sidecar (with the precise capture date, GPS, caption, and people names) is in part 8, the photo used to land in your library with a "Recovered" or "Marked" date because PDR analyses one part at a time. v2.0.13 pre-scans every Takeout part's sidecars into a cross-part cache, then runs an Enrichment pass that retroactively upgrades the photos: writes the precise EXIF date, the GPS coordinates, the description (caption), and seeds suggested face names — all without touching the photo bytes. The pass is strictly additive: it never overrides a date you've manually set, a face you've named, or a caption you've typed. Found and runs from the Library Drive Manager, under "Takeout metadata".

Photo captions:

Right-click any photo in Albums, By Date, or Search & Discovery and pick "Add caption" to type a note that travels with the photo. The caption is saved in your PDR library and (when possible) written into the photo's EXIF ImageDescription + XMP dc:description, so it exports alongside the file when you copy it out. A small gold chat-bubble indicator appears in the corner of any captioned thumbnail, and hovering anywhere on the photo reveals the caption in a gold tooltip — distinct from PDR's lavender system tooltips so personal notes look personal. In Albums, a "Captioned only" filter chip in the header lets you narrow the grid to just the photos with captions. The viewer also shows a dedicated caption row below the filmstrip.

By-Date timeline rebuild:

The month drilldown used to be a flat wall of photos with no day separators. v2.0.13 adds per-day sticky headers (long-form: "Tuesday, 7 February 2022 — 23 photos"), a floating "current day" pill that follows the scroll position so you always know what date you're looking at, and two prev/next-day arrows in the bottom-right corner (also bound to PageUp / PageDown on the keyboard) so you can jump cleanly between days instead of free-scrolling.

Library Drive Manager:

The Takeout metadata section now shows a "Last enriched X ago — N files upgraded" line under each scanned export, so you can see at a glance when the pass last ran without opening the modal.

Album thumbnail chooser:

Right-click any photo in an open album and pick "Set as album thumbnail" to override the auto-picked first-photo cover. The album list updates immediately.

Robustness fixes:

• Enrichment dedup — when the same photo has been Fixed twice from different sources (one with a sidecar, one without), PDR now SHA-256-compares the two copies and cleans up the redundant one. Same-size copies whose only difference is EXIF metadata get a smarter image-data hash so they're recognised as the same photo.

• Custom monthly thumbnail — setting "Set as monthly thumbnail" on a photo now refreshes the By-Date grid immediately. Previously the tile kept showing the auto-picked image until the page was reloaded.

• Caption save no longer freezes the window — the EXIF write runs in the background instead of holding the IPC open while exiftool's subprocess is busy with another caller.

This update will install automatically the next time you close PDR.
